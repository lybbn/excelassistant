说明：

本软件主要把xls得excel文件导入到软件列表，通过列表搜索结果再导出xls得excel文件

软件截图：

[excel助手][(excel文件助手1.png)]

使用方法：

（1）先点击【导入xls文件】（非xls文件后缀得，请重新保存文件为xls后缀的文件）

（2）导入后如果有多个工作表，可选择需要导入的工作表序列号，软件会读取excel数据到控制台，并显示再原始数据区域（数据量大的话需要等一会，等待导入完毕）

（3）通过搜索框【搜索】需要寻找的文本内容（数据量大的话需要多等待一会），搜索出的结果会出现再过滤数据的窗口中，点击【导出】保存excel文件

注意：

本软件为绿色无毒软件，如杀毒软件报毒可点击信任本软件，继续使用
